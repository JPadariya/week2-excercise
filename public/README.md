# Public Assets

This directory contains static files that are publicly accessible.

## Structure

- `css/` - Stylesheets
- `js/` - Client-side JavaScript
- `images/` - Images and icons
- `fonts/` - Custom fonts

## Guidelines

- Optimize images for web (compress, use appropriate formats)
- Minify CSS and JS for production
- Use semantic file names
- Keep assets organized by type
